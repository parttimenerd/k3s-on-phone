OPTIONAL: Enhanced AI with Gemma Language Model

To enable advanced natural language AI responses:

1. Download a Gemma model in .task format for MediaPipe
2. Rename it to "gemma.task" 
3. Place it in this assets/ folder
4. Rebuild the app

The app will automatically detect gemma.task and enable enhanced AI features.

Without Gemma:
- Structured vision analysis responses
- Object detection and classification
- Image embeddings and similarity

With Gemma:
- Natural language explanations
- Advanced reasoning about images
- Conversational AI responses

Note: Adding Gemma will increase APK size by ~2GB
